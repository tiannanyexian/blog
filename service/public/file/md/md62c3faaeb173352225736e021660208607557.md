# VUE学习笔记

1. v-bind动态绑定标签属性的值   简写 : 

   ```vue
   <!--
   根据数据 isRed 的真假值，来决定有没有"red"这个名字。是否添加这个类
   -->
   <div :class="{red:isRed}"></div>
   
   <div :class="classObject"></div>
   <!--
   数据
   {
   	classObject:{
   		aaa : true,
   		bbb : false
   	}
   }
   
   渲染为
   <div class="aaa"></div>
   -->
   ```
