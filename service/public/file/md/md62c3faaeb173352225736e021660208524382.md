### 		运算符：

```js
		数学运算符: + - * %
            与字符串相加 + 表示的是拼接 "1"+1=11
            - * % 则默认双方都为number类型 "1"-1=0
            ** 表示的是^  2**2=4 => 2^2=4
            % 是取模的意思,取的是余数 10%2 答案是0 10%3 答案是1 10%100 答案是10 对于小的数%大的数的时候 直接取%前面的数							
		前置++ --  先赋值,再运算 (表示参与运算,并修改原值)
		后置++ --  先运算,再赋值 (表示不参与运算,只修改原值)
		
		赋值运算符: = += -= /= *= **= %=
	
		比较运算符: < > <= >= == != === !== 比较的结果是true 或 false
            字符串的比较 根据ASCII表的排序来比较的 一个一个字符的比较 "20">"100" 2在ASCII表中排50,1排49 所以"20">"100"
            对象的比较 比较的是内存地址
            != ==比较 比较时会转换成向同类型在比较 "1"==1 true
            !== ===比较 比较时不会发生类型转换 "1"===1 false
		
		逻辑运算符: &&(与) ||(或) !(非)
	
		优先级: 
           () > 逻辑非(!) > 一元运算符(++ --) > 次幂(**) > 数学运算符(+ - * %) > 比较运算符(< > <= >= == != === !== ) > 逻辑与>或(&&>||) 
```

```js
		class类 必须通过new启动 

		class创建一个类,类名为Person
			constructor 是一种用于创建和初始化class创建的对象的特殊方法。通过new创建class类 自动执行constructor函数
	
		class Person{
            //各种属性 方法
            constructor(name,age) {//new class类名时进行的传参,在constructor形参中接收
                this.name=name;//this传参 this就是class类,指向Person
                this.age=age;
            }
            add(){
                console.log("我是Person中的add方法");
                this.#del();
            }
            #del(){//方法以#开头表示私有方法,只能内部调用
                console.log("我是Person中的del方法");

            }
            static updata(){//方法以static开头表示静态方法,内部和外部都不能调用,只能通过类名调用
                console.log("我是Person中的updata方法");
            }
        };

            let result = new Person();//new实例化,获取类的对象 在new实例化后，其成员被实例化，实例对象方可调用。
            result.add();
            //Person.prototype.add();
            Person.updata();

                原型 没有放在constructor函数里面的,会放在原型上   add() del()
                类名 Person
                实例 result 会继承它的原型
                类名=>原型  Person.prototype => 选择到原型
                实例=>原型  __proto__ => 选择到原型

            子类继承父类
                函数或者属性名重复,就近原则

                 super(name,age); //调用父类,也可向父类传参
```
